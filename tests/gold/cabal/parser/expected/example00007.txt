  DESCRIPTION
    PsiElement(DESCRIPTION_KEY)('Description')
    PsiElement(COLON)(':')
    FREEFORM
      PsiElement(FREEFORM_LINE)('Pandoc is a Haskell library for converting from one markup')
      PsiElement(FREEFORM_LINE)('format to another, and a command-line tool that uses')
      PsiElement(FREEFORM_LINE)('this library. It can read several dialects of Markdown and')
      PsiElement(FREEFORM_LINE)('(subsets of) HTML, reStructuredText, LaTeX, DocBook,')
      PsiElement(FREEFORM_LINE)('MediaWiki markup, TWiki markup, Haddock markup, OPML,')
      PsiElement(FREEFORM_LINE)('Emacs Org-Mode, txt2tags, Word Docx, ODT, and Textile, and')
      PsiElement(FREEFORM_LINE)('it can write Markdown, reStructuredText, XHTML, HTML 5,')
      PsiElement(FREEFORM_LINE)('LaTeX, ConTeXt, DocBook, OPML, TEI, OpenDocument, ODT,')
      PsiElement(FREEFORM_LINE)('Word docx, RTF, MediaWiki, DokuWiki, Textile, groff man')
      PsiElement(FREEFORM_LINE)('pages, plain text, Emacs Org-Mode, AsciiDoc, Haddock markup,')
      PsiElement(FREEFORM_LINE)('EPUB (v2 and v3), FictionBook2, InDesign ICML, and several')
      PsiElement(FREEFORM_LINE)('kinds of HTML/javascript slide shows (S5, Slidy, Slideous,')
      PsiElement(FREEFORM_LINE)('DZSlides, reveal.js).')
      PsiElement(FREEFORM_LINE)('.')
      PsiElement(FREEFORM_LINE)('In contrast to most existing tools for converting Markdown')
      PsiElement(FREEFORM_LINE)('to HTML, pandoc has a modular design: it consists of a set of')
      PsiElement(FREEFORM_LINE)('readers, which parse text in a given format and produce a')
      PsiElement(FREEFORM_LINE)('native representation of the document, and a set of writers,')
      PsiElement(FREEFORM_LINE)('which convert this native representation into a target')
      PsiElement(FREEFORM_LINE)('format. Thus, adding an input or output format requires')
      PsiElement(FREEFORM_LINE)('only adding a reader or writer.')
